# PowerSync Collection

**Package:** `@tanstack/powersync-db-collection`

**Current Version:** 0.1.20

## Description

The PowerSync Collection package provides SQLite-based offline-first synchronization capabilities for TanStack DB. It enables seamless integration with [PowerSync](https://powersync.com/), allowing your application to work offline with local SQLite storage while automatically synchronizing data when connectivity is available.
