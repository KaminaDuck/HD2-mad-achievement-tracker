# LocalOnly Collection Version History

LocalOnly Collection is part of the core `@tanstack/react-db` package and follows its versioning. The current version is **0.1.60**.

## Version Notes

Since LocalOnly Collection is bundled with `@tanstack/react-db`, it does not have its own separate release cycle. All changes to LocalOnly Collection functionality are released as part of the main react-db package updates.

For the complete version history including all changes to LocalOnly Collection, refer to the [main TanStack DB versions.md](../../versions.md).
