# @tanstack/react-db Version Log

Version history for the `@tanstack/react-db` package.

## Current Version

### v0.1.60 (Current)

The current stable version of the React adapter for TanStack DB.

This is the primary adapter for using TanStack DB with React applications, providing:

- `useLiveQuery` - Live queries with automatic updates
- `useLiveSuspenseQuery` - Suspense-compatible live queries
- `useLiveInfiniteQuery` - Paginated live queries
