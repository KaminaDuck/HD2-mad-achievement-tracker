# @tanstack/angular-db

Angular adapter for TanStack DB providing observables and signals for reactive live queries.

**Current Version:** 0.1.42

## Overview

The `@tanstack/angular-db` package provides Angular-specific bindings for TanStack DB. It enables reactive live queries using Angular's native patterns including observables and signals.

## Installation

```bash
npm install @tanstack/angular-db
```

## Key Features

- **Observables**: RxJS Observable-based API for live queries
- **Signals**: Angular signals support for reactive data
- **Live Queries**: Automatic updates when underlying data changes
- **Type Safety**: Full TypeScript support with strong typing
